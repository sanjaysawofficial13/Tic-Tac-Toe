# Tic-Tac-Toe

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.
Play against another player in a classic 3x3 grid with real-time win detection.

🧠 Game Rules

The game is played on a 3×3 grid.

Player X goes first, followed by Player O.

The first player to align 3 marks horizontally, vertically, or diagonally wins.

If all 9 cells are filled and no winner is found, the game ends in a draw.
